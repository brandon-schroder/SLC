# LS89 — VKI LS-89 transonic turbine nozzle guide vane (Arts et al.)

**Source:** Arts & Lambert de Rouvroit, *Aero-Thermal Performance of a
Two-Dimensional Highly Loaded Transonic Turbine Nozzle Guide Vane* (VKI
TN-174 / J. Turbomach. 1992, DOI 10.1115/1.2927978). In the
"Turbomachinery: Test Cases" NotebookLM notebook; values below extracted
verbatim via the notebook (2026-07-15).

**Role:** the standard transonic HP-turbine cascade — the V6 cascade-level
measured-data check of the Ainley exit-angle rule and the K-O loss chain
(`model-readiness` gate #1; the turbine analogue of the TR-1368 pass).

## Extracted values

- Chord 67.647 mm; pitch/chord g/c = 0.850; stagger 55.0° (from axial);
  throat/pitch **o/g = 0.2597**; TE radius r_TE/c = 0.0105
  (t_TE = 2·r_TE → t_TE/o = 0.0951); axial inflow (α1 = 0).
- Loss = **energy loss coefficient** ζ₂ = 1 − V₂²/V₂,is² (exact
  compressible p-ratio form quoted in the paper).
- **Measured at M2is = 1.0:** total ζ₂ ≈ **2.25%**, decomposed ≈ 1.0%
  boundary-layer + 0.75% trailing-edge + 0.5% shock + ~0 base pressure.
  Re₂,is 0.5–2.0×10⁶, loss insensitive to Re in that range. The
  loss-vs-M2is characteristic (M2 0.7–1.1) is Fig. 16 — a figure, not
  tabulated; exact per-Mach values "available upon request".
- Exit flow angle ≈ the gauging angle sin⁻¹(o/g) = 15.05° from tangential
  (= 74.95° from axial); detailed per-Mach angle calibration not tabulated.

## Comparison vs the coded closures (pinned: `tests/test_v6_ls89.py`)

Closure-level (2-D stationary cascade — pure K-O functions mirroring
`KackerOkapuuLoss.evaluate` for a nozzle at midspan; no machine solve):

- **Exit angle:** `throat_exit_angle(0.2597)` = 74.95° — coincides with the
  paper's gauging angle to 0.1° (the arccos(o/s) M2=1 gauge rule evaluated
  exactly at its design condition, M2is = 1.0).
- **Total loss:** predicted Y 0.0464 → energy-ζ **0.0303 vs measured
  0.0225 (+35%)**. K-O over-predicting a modern high-efficiency vane by
  30–50% is the documented behaviour of the 1982 mean-line method
  (calibrated on older hardware, ±1.5% stage-η target). Band pinned
  [0.7×, 1.7×] — catches a regression to the pre-calibration constants
  (~3× high) or a reference-frame error.
- **TE component:** K-O ζ_TE(t/o=0.095) ≈ 1.9% vs the rig's measured TE
  share ~0.75% — the K-O TE curve carries most of the over-prediction here
  (calibrated on thicker-TE hardware); recorded, bounded in test.
- **Shock:** the K-O shock term is an *inlet* (LE) component and correctly
  reads ~0 at M1 ≈ 0.15; the measured 0.5% is an *exit/TE* shock system the
  method does not model separately at M2 ≤ 1 — recorded model boundary.
