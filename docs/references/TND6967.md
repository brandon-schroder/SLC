## Measured agreement (2026-07-16; pinned in `tests/test_v6_tnd6967.py`)

Tier-1 meanline at the measured equivalent flow (2.004 kg/s), 800 outer
iterations (the 4-row Picard chain at `closure_relax` 0.25 is slow, not
unstable), validity 0.99:

| quantity | code | measured | Δ |
|----------|------|----------|---|
| η_tt | 0.926 | 0.93 | −0.4 pt |
| PR_tt | 3.13 | 3.765 | −17% |
| work [J/g] | 74.6 | 84.9 | −12% |

**Findings.**
1. **η_tt within half a point** through four K-O rows end-to-end — and the
   set carries no tip-clearance loss (rotors ran ~0.9% clearance/height),
   so the internal-loss level is honestly ~1–1.5 pt pessimistic: inside
   K-O's own ±1.5 pt stage-η target.
2. **The PR/work gap re-diagnosed (2026-07-16, row-throat check landed):
   near-choke sensitivity, not a capacity error.** The §6.6 row-throat
   capacity check measures the model choking at ~2.02 kg/s eq (annulus;
   rotor-1's throat 2.06, stators 2.14–2.19 at cd = 1) vs the rig's
   measured 2.03–2.05 — **within ~1%**. (The earlier "capacity ≈ 2.19 vs
   2.03" line was a hand stator-gauge estimate, superseded.) Both rig and
   model sit within 1–2% of choke at 2.004 kg/s, where PR-vs-mdot is
   near-vertical, so a matched-MDOT comparison amplifies the ~1% capacity
   difference into the −17% PR read.

   **Matched-PR comparison RUN (2026-07-16,
   `test_matched_pr_backpressure_comparison`):** imposing the Table-I
   equivalent total-to-static PR (p_exit = p0_in/4.640) via
   `BackPressureSpec` + Newton, seeded from the converged matched-mdot
   state:

   | quantity | model | measured | Δ |
   |---|---|---|---|
   | ṁ eq | 2.011 | 2.004 | +0.35% |
   | PR_tt | 3.718 | 3.765 | −1.2% |
   | work | 83.87 | 84.90 J/g | −1.2% |
   | η_tt | 0.926 | 0.93 | −0.4 pt |

   **In the natural frame the machine agrees with the rig to ~1% across
   the board** — no additional turning/loss deficit hides behind the
   matched-mdot read. The initially-observed **seed-dependent fixed
   points** were root-caused same day: the fresh mdot-2.0 seed's solve
   converged station 7 (rotor-2 LE) onto the **supersonic continuity
   root** (M_m 1.997, Vm 452 m/s; root pair proven on the frozen fields —
   subsonic 112, capacity peak ~250, supersonic 452), which the
   positivity-only Newton guard accepted and the closure lag locked in.
   **Fixed** by the §6.3 branch-preserving trial guard
   (`newton._safe_residual`, `_BRANCH_DELTA = 0.05`): trials may approach
   M_m = 1 but not jump across it relative to the seed's per-station
   branch — nearby seeds now land the identical physical fixed point; far
   seeds fail TYPED (Newton is local); the supersonic-branch handoff is
   *protected* (its landed seed classifies supersonic and can no longer
   fall back). Regression:
   `test_backpressure_newton_stays_on_seed_branch`. The closure lag still
   rings at ~1e-6 near choke (benign; `tol_closure` loosened in the
   pinned tests).
3. **Tier-2 spanwise is OPEN**: at measured flow the free-vortex hub
   chokes its streamtube (CHOKE_LIMITED); reduced flows hit
   NUMERICAL_FAILURE — the 4-row spanwise closure-lag chain joins the
   recorded multi-row spanwise robustness items. Meanline carries the
   validation.

## Multi-speed map — Fig. 17(a) (2026-07-17; `MEASURED_MAP`,

The two-stage overall performance map (equivalent specific work vs the
equivalent mass-flow–speed parameter εwω/δ, speed lines 30–110%, PR_tt and

ticks; **the plotted equivalent-design dot reproduces its published
coordinates (3.22×10³, 84.9) exactly — the calibration control.** Four
speed-line ∩ PR-contour intersections read (±1.5 J/g), with equivalent
flow recovered from the abscissa (x/ω_eq):

| speed | PR_tt | work map → model | flow map → model |
|---|---|---|---|
| 90% | 3.4 | 79.0 → 78.0 (−1.3%) | 2.00 → 2.018 (+0.9%) |
| 90% | 3.0 | 71.0 → 71.4 (+0.5%) | 1.99 → 2.011 (+1.0%) |

| 50% | 2.2 | 46.9 → 47.8 (+1.9%) | 2.03 → 2.054 (+1.2%) |

Model values from interpolating the warm-chained classical characteristic
to each map PR_tt (work(PR) is smooth even where mdot(PR) is
near-vertical; the first BP-secant attempt from a far ṁ=1.9 seed
measurably fails — Newton overshoots choke — the matched-PR frame near
choke *requires* an adjacent seed, consistent with the §6.3 branch-guard
findings). **The design-point ~1% agreement holds across the map**: work
within ±2.2% and flow within ~+1% (the known capacity offset) across
50–100% speed and PR 2.2–3.765. The rig's choked-flow structure is
reproduced *including its trend*: map flow rises ~1.5% from 100%→50%
speed (2.00→2.03); the model's rises the same way (2.02→2.055).
η contours (0.90–0.925 over this region) are consistent with the model's
0.926 but were not digitized point-by-point (the closed contours are too
coarse to localize).
